Arrays and Hashes

Wednesday Jan 28, 2015

Arrays

Think of an array as an empty bucket. Whenever you want it, you just pick it up and toss whatever you'd like in it. In Ruby, all you need to do is name the bucket and either leave it as empty for future use or name the bucket and say what you want to put it in it. You can put strings, numbers, more arrays, hashes, etc. Arrays can hold pretty much anyhting. You can create an empty array one of two ways:

Method 1:
new_array = []
Method 2:
new_array = Array.new()

Similarly, you can add values to an array at the same time as you create it. Like so: 

                
                new_array = ["cat", "hat", "bat"]

To access an item in your array, you type the number where the item is located in the array in brackets after the array. Remember that arrays are numbered starting from 0. So in the above example, to access "cat" you would type: 

 
                > new_array = ["cat","hat", "bat"]
                => ["cat", "hat", "bat"]
                > new_array[0]
                => "cat"
                > new_array[1]
                => "hat"
                > new_array[2]
                => "bat"
                > new_array[3]
                => nil

Arrays come with their own set of methods from the class Array. In addition, you can add two arrays as well as subtract them. Adding arrays will put all the values of the two arrays into one array. Subtracting one array from a second array will return all the items of the first array that were not in the second array. Lastly, you can find all common items of two arrays by joining them with an ampersand (&) likes so:
>[1,2,3]&[2,3,4]
=> [2,3]

Arrays have many more methods that come with their class. Check them out.

Hashes

Hashes are very similar to arrays in that they both store data. The main difference is that hashes store their data through the use of "keys" and "values" while arrays store their data by using indices. So all data in a hash is a value and every value has a key. Making a hash is similar to making a new array. Only difference is the use of squiggly brackets {} instead of square [] that are used for arrays: 


                > new_hash = Hash.new
                => {}
                > new_hash = {}
                => {}

To match a value to a key when creating a new hash, you use the hash rocket (=>) like so: 


                favorite_athletes = {"football" => "Tom Brady", "basketball" => "Michael Jordan", "baseball" => "Derek Jeter"}

Hashes are good to use when you have more data or data within data that you need to store.
